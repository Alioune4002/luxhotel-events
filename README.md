# LuxHotel Events

Site web pour un hôtel et événementiel de luxe, construit avec **React**, **Django**, et **Tailwind CSS**. Basé sur un template Figma, il inclut :
- Interface responsive avec slider interactif.
- API REST pour gérer chambres, événements, et réservations.
- Images haute qualité pour un design chic et moderne.
- Formulaires de contact et réservation fonctionnels.
- Déploiement sur Vercel (frontend) et Heroku (backend).
